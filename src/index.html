Notes to the graders

1. This is a minecraft world with the character Steve, several pigs, small houses, and a fence.
2. The mouse movement for the camera can be a bit buggy, but it works better when you are moving with the WASD keys.
3. You can use the left/right arrows for turning, and the up/down arrows for moving upward/downward.
4. You can not go outside the brown outer wall of the world.
5. I used copilot to help autofill the matrix translations and other repetitive code to speed up development.
6. I referenced the professor's code for the initial setup and also used the textbook.
7. The story is that Steve is managing a farm with pigs.
8. I used several textures: one for the sky and several from images of minecraft materials
